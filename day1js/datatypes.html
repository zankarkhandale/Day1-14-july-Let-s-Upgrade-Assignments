Data Types Intro 

            Javascript has Dynamic Data Types that means same variables can be used to hold different data types.

            Javascript variables can hold many data types :numbers ,strings,objects and more

            Javascript data types there are two types Primitve and non-primitive

In primitive there are:Number, Bingint,String, Boolean, Null, Undefined, typeof
